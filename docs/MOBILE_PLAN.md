> 版本: v1.0 | 日期: 2026-06-19

---

## 三、方案详解

### 方案 A：响应式 H5（✅ 当前推荐）

**现状**：已完成，立即可用。

**传播路径**：
```
小红书/抖音/微博 → 链接/二维码 → 微信内置浏览器打开 H5
    → 完成测试 → 分享海报（已有Canvas生成） → 裂变
```

**优势**：
- ¥0 费用，无需审核，随时更新
- 不受平台限制（微信/微博/浏览器/任何 App 内嵌均可）
- 分享海报已有 Canvas 生成功能

**局限**：
- 不能使用微信支付（需要小程序）
- 搜索引擎收录需做 SSR/预渲染

---

### 方案 B：PWA（推荐优先实施）

在方案 A 基础上，加一个 `manifest.json` + Service Worker：

| 功能 | 效果 |
|------|------|
| 桌面图标 | 用户可"添加到主屏幕"，像 App 一样打开 |
| 离线可用 | 缓存静态资源，无网络也能浏览 |
| 推送通知 | Web Push API（需 HTTPS） |
| 启动画面 | 自定义 Splash Screen |

**实施**：1-2 天，¥0。

---

### 方案 C：微信小程序（个人）

**你需准备**：

| 步骤 | 说明 | 时间 |
|------|------|------|
| 1. 注册 | [mp.weixin.qq.com](https://mp.weixin.qq.com) 选择"个人"主体 | 1 天 |
| 2. 域名备案 | 后端 API 域名必须已备案（个人也可备案） | 15-20 工作日 |
| 3. HTTPS | 后端 API 需 HTTPS（Let's Encrypt 免费） | 1 天 |

**个人小程序限制**：
- ❌ 无法使用微信支付
- ❌ 无法使用部分高级 API（如获取手机号）
- ❌ 部分类目受限

---

### 方案 D：微信小程序（企业）

**费用**：认证费 ¥300/年 + 服务器 ¥50-200/月

**额外需准备**：
- 营业执照
- 企业对公账户
- 法人身份验证

**优势**：可开通微信支付，可发布付费报告。

## 四、推荐路径

```
当前（0 天）
    │
    ├─ 方案 A：响应式 H5（✅ 已就绪）
    │      └─ 部署后立即在微信/浏览器中分享传播
    │
    ├─ 下一步（1-2 天）
    │      └─ 方案 B：PWA 化（¥0，离线+桌面图标）
    │
    └─ 用户规模验证后（3-6 个月）
           └─ 方案 D：企业小程序 + 微信支付（¥300/年）
```

### 现阶段建议

**不做小程序，优先 PWA 化（1-2 天，¥0）**。理由：

1. 响应式 H5 在微信里已能用，扫码即开
2. PWA 让用户能"装到桌面"，体验接近 App
3. 小程序审核 + 备案至少耽误 3 周
4. 用户积累到一定规模再做小程序更划算

## 六、生产部署简易方案（¥50-200/月）

| 方案 | 费用 | 说明 |
|------|------|------|
| **腾讯云轻量应用服务器** | ¥50/月 | 2C2G，Nginx 反向代理 + Node |
| **阿里云 ECS** | ¥70-100/月 | 类似 |
| **Vercel（前端） + 腾讯云（后端）** | 前端免费 + 后端 ¥50/月 | 最佳组合 |
| **Cloudflare Pages（前端） + 腾讯云（后端）** | 前端免费 + 后端 ¥50/月 | 全球 CDN |

推荐：**Vercel 托管前端静态文件（免费）+ 腾讯云轻量服务器跑后端 API（¥50/月）**。
