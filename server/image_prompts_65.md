# 65型人格几何图像AI生成提示词

## 推荐大模型

### 国内首选：
1. **通义万相 (Tongyi Wanxiang)** — 阿里云出品
   - 优势：几何抽象风格表现出色，支持中英文提示词，色彩还原准确
   - 接入：阿里云百炼平台 / DashScope API
   - 模型：`tongyi-wanxiang-v1.0` 或 `wanx2.0`
   - 成本：约 ¥0.06-0.12/张

2. **即梦 (Dreamina / Jimeng)** — 字节跳动出品
   - 优势：概念艺术和抽象风格渲染力强，对几何构成理解到位
   - 接入：即梦网页版 / 火山引擎 API
   - 模型：`jimeng-t2i-v2`
   - 成本：约 ¥0.05-0.10/张

3. **文心一格 (ERNIE-ViLG)** — 百度出品
   - 优势：中文语义理解最优，对中国风抽象元素把握细腻
   - 接入：百度智能云 / 文心一言平台
   - 成本：会员制 or 按量计费

### 推荐优先使用：**通义万相**（几何抽象+色彩+成本综合最优）
### 备选方案：**即梦**（如果追求更强的艺术感和创意表现力）
