## 一、题目类型体系

### 1.1 MBTI标准题型分类

| 题型 | 说明 | 适用场景 | MBTI-PRO使用 |
|------|------|---------|-------------|
| 二选一迫选题 | 强制在两个对立陈述中选择 | 官方专业版(Step I)核心题型 | 不采用 |
| 李克特5级量表 | 5个同意程度选项 | 大众网络版主流题型 | **80题（主观题）** |
| 场景行为题 | 模拟真实场景选择行为 | 高占比实操题型 | 融入李克特题 |
| 自我特质描述题 | 自我评价特质倾向 | 辅助验证题型 | 融入李克特题 |
| 客观选择题 | 4选1，有标准答案 | T/F决策维度 | **20题** |

### 1.2 为什么MBTI-PRO不采用二选一迫选题？

1. 项目计划书明确要求主观题80题 + 客观题20题
2. 李克特5级量表能更好地测量倾向强度（而非仅方向）
3. 5级量表配合三级分类（E/A/I）在数学上更合理
4. 迫选题强制二分与MBTI-PRO"中间值"理念相冲突

## 二、各维度题目设计参考

### 2.1 E/I 维度（能量来源）- 25题

**子维度分布（各5题）：**

#### 社交偏好 (Social Preference)
参考题目方向：
- "参加大型聚会后，我感到精力充沛"
- "比起认识新朋友，我更喜欢和老朋友深入交流" (R)
- "我喜欢成为人群中的焦点"

#### 独处需求 (Solitude Need)
参考题目方向：
- "长时间社交后，我需要独处来恢复精力" (R)
- "独自工作比团队协作效率更高" (R)
- "我享受一个人的时光，不会感到无聊" (R)

#### 表达方式 (Expression Style)
参考题目方向：
- "我倾向于先说再想，边说边理清思路"
- "我更习惯先想好再开口" (R)
- "在会议上我通常是主动发言的那个人"

#### 注意力方向 (Attention Direction)
参考题目方向：
- "我更容易注意到周围环境的变化"
- "我经常沉浸在自己的想法中而忽略外界" (R)
- "相比外部活动，我更喜欢探索内心世界" (R)

#### 精力恢复 (Energy Recovery)
参考题目方向：
- "忙碌一周后，我更想约朋友出去放松"
- "压力大的时候，我需要一个人静静" (R)
- "与人交流会让我感到充电"

> (R) 标记为反向计分题

### 2.2 S/N 维度（认知方式）- 25题

**子维度分布：**

#### 信息关注点 (5题)
- "我更关注事情的细节和具体步骤"
- "我善于发现事物之间的模式和联系" (R)

#### 解决问题 (5题)
- "遇到问题时，我倾向于参考过去的经验"
- "我喜欢尝试新的方法，即使没有先例" (R)

#### 思维方式 (5题)
- "我更相信亲眼所见的事实和数据"
- "我经常做白日梦或进行天马行空的想象" (R)

#### 学习偏好 (5题)
- "我通过反复练习来掌握新技能"
- "我更喜欢先理解整体框架再学习细节" (R)

#### 未来规划 (5题)
- "我专注于做好眼前的事情"
- "我经常思考未来的各种可能性" (R)

### 2.3 T/F 维度（决策方式）- 5题主观 + 20题客观

**主观题子维度（5题）：**
- 价值观判断
- 人际情境决策
- 冲突处理方式
- 公平与关怀权衡
- 规则与例外判断

**客观题子维度（20题）：**

| 类别 | 题数 | 示例方向 |
|------|------|---------|
| 逻辑推理 | 8题 | 演绎推理、归纳推理、逻辑一致性判断 |
| 数据分析 | 6题 | 图表数据解读、概率判断、统计推理 |
| 决策评估 | 6题 | 多因素权衡、最优方案选择、利弊分析 |

**客观题示例：**
```
【逻辑推理题】
如果所有A都是B，所有B都是C，那么以下哪个陈述一定成立？
A. 所有C都是A
B. 有些A不是C
C. 所有A都是C ← 正确答案
D. 有些C不是A

【数据分析题】
某公司Q1-Q4的营收分别为100/120/90/150万，利润率为20%/18%/25%/22%。
请计算全年总利润约为多少？
A. 94.6万
B. 100.2万 ← 正确答案
C. 85.3万
D. 110.5万

【决策评估题】
一个产品开发项目面临三种方案：
方案A：成功概率60%，成功后收益1000万，失败损失200万
方案B：成功概率85%，成功后收益500万，失败损失100万
方案C：成功概率40%，成功后收益2000万，失败损失500万
从期望值最大化角度，应选择：
A. 方案A (期望=520万) ← 正确答案
B. 方案B (期望=410万)
C. 方案C (期望=500万)
D. 无法判断
```

### 2.4 P/J 维度（生活态度）- 25题

**子维度分布：**

#### 计划性 (5题)
- "我凡事喜欢提前做计划"
- "我倾向于见机行事，不太喜欢被计划束缚" (R)

#### 秩序感 (5题)
- "我的工作和生活空间通常整洁有序"
- "一定的混乱不会影响我的效率" (R)

#### 时间管理 (5题)
- "我总是提前完成截止日期前的任务"
- "截止日期的压力能让我更有动力" (R)

#### 决策速度 (5题)
- "做决定时我倾向于快速拍板"
- "我喜欢保留所有选项，直到最后一刻" (R)

#### 变化应对 (5题)
- "突然的计划变动会让我感到不适"
- "我享受生活中的不可预测性" (R)

## 三、题目质量审核清单

审核每道题目时检查：

### 内容质量
- [ ] 题目表述清晰，无歧义
- [ ] 使用日常语汇，避免专业术语
- [ ] 适配中文文化语境（非翻译腔）
- [ ] 无引导性（不暗示"正确答案"）
- [ ] 一道题只测一个维度

### 技术质量
- [ ] 正向/反向方向标注正确
- [ ] 选项表述与题目匹配
- [ ] 子维度与主维度归类正确
- [ ] 难度等级标记
- [ ] 与同维度其他题不重复

### 客观题特别检查
- [ ] 正确答案确实唯一且正确
- [ ] 干扰选项具有迷惑性但明确错误
- [ ] 题目不依赖专业知识
- [ ] 计算题确保数字准确

## 五、出题优先级

基于开发流程，建议按以下顺序出题：

1. **E/I 维度**（最易理解，最易出题）→ 第一周
2. **S/N 维度**（与E/I难度相当）→ 第一周
3. **P/J 维度**（可参考现成题目多）→ 第二周
4. **T/F 主观5题**（题量少）→ 第二周
5. **T/F 客观20题**（难度最高）→ 第二-三周
