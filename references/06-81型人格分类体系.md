## 一、分类编码体系

### 1.1 编码规则

```
人格类型代码 = [能量来源] [认知方式] [决策方式] [生活态度]

每个位置可取3个值，共 3×3×3×3 = 81 种组合
```

### 1.2 各位置取值

| 位置 | 维度 | 取值1(正向) | 取值2(中性) | 取值3(负向) |
|------|------|-----------|-----------|-----------|
| 1 | 能量来源 | E (外向) | A (平衡) | I (内向) |
| 2 | 认知方式 | N (直觉) | B (均衡) | S (实感) |
| 3 | 决策方式 | T (思考) | C (复合) | F (情感) |
| 4 | 生活态度 | J (判断) | D (动态) | P (感知) |

### 1.3 中间值命名与含义

| 代码 | 英文全称 | 中文名称 | 核心含义 |
|------|---------|---------|---------|
| **A** | Ambivert | 平衡型(能量) | 灵活切换社交与独处模式，在不同场合展现不同倾向 |
| **B** | Balanced | 均衡型(认知) | 兼顾现实细节与抽象可能，务实与创新并重 |
| **C** | Composite | 复合型(决策) | 决策时综合考虑逻辑分析与人际因素 |
| **D** | Dynamic | 动态型(态度) | 有计划也有弹性，能根据情境切换结构化与灵活模式 |

## 二、完整81型人格编码列表

### 2.1 总览矩阵

四个位置分别有3个选项：
- 位置1: E / A / I
- 位置2: N / B / S
- 位置3: T / C / F
- 位置4: J / D / P

3×3×3×3 = 81 种

### 2.2 与传统16型的关系

**传统16型在81型中的定位：**

传统16型 = 位置1∈{E,I} × 位置2∈{N,S} × 位置3∈{T,F} × 位置4∈{J,P}

这16型代表了"极端化"的人格画像，在81型体系中它们是四个维度全部落在两极的类型。

| 传统类型 | 在81型中代码 | 说明 |
|---------|-------------|------|
| INTJ | INTJ | 四个维度全负向极端 |
| ENTP | ENTP | E正向+N正向+T正向+P负向 |
| ... | ... | 全部16型保持不变 |

**新增65型 = 81 - 16**

新增类型中至少有一个维度为中间值。


## 三、中间值维度对各人格的影响

### 3.1 能量平衡型 (A) 人格特征

**A型人的共同特点：**
- 社交灵活性高：能享受聚会也能享受独处
- 环境适应性强：根据不同场合调整行为模式
- 能量来源多元：既能从外部获取也能从内部获取能量
- 相比E型更内敛，相比I型更开放

**A型在传统分类中的典型"矛盾"：**
- 测试结果在E/I之间反复摇摆的用户
- 既喜欢团队协作也喜欢独立工作的职场人
- 在社交后不需要特别长的恢复期，但也不从社交中获取主要能量

### 3.2 认知均衡型 (B) 人格特征

**B型人的共同特点：**
- 信息处理全面：既有细节把控又有大局观
- 思维灵活转换：能在具体和抽象之间自由切换
- 实用创新并重：既能落地执行也能头脑风暴

**B型常见表现：**
- 既能做详尽的数据分析，也能做战略规划
- 学习时既看理论框架也重视实操练习
- 在S/N维度测试中接近零分的人

### 3.3 决策复合型 (C) 人格特征

**C型人的共同特点：**
- 决策平衡：考虑逻辑也考虑人情
- 情境判断力强：根据具体情况选择偏逻辑或偏情感的决策方式
- 适应性高：不容易因单一决策风格导致问题

**C型与T型和F型的区别：**
- 不像T型那样"冷漠无情"
- 不像F型那样"感情用事"
- 在管理岗位和需要综合判断的职位中常见

### 3.4 态度动态型 (D) 人格特征

**D型人的共同特点：**
- 灵活有度：需要结构时能制定计划，需要变化时能随机应变
- 压力管理好：不会因计划变动而焦虑，也不会因缺乏计划而迷失
- 效率与创造力的平衡

**D型常见表现：**
- 工作中会做计划但不死守计划
- 既享受完成任务的成就感，也享受过程中的探索
- P/J维度测试分数接近零的人

## 四、描述文案生成策略

### 4.1 模块化构建方法

81型人格描述采用 **"基础框架 + 维度组合"** 的模块化方法：

```
人格描述 = 
  通用框架（50字） +
  能量维度描述（E/A/I模块,100-150字） +
  认知维度描述（N/B/S模块,100-150字） +
  决策维度描述（T/C/F模块,100-150字） +
  态度维度描述（J/D/P模块,100-150字） +
  组合效应描述（特定组合的影响,100-200字） +
  职业建议（基于四维组合,5-8岗位） +
  成长建议（通用+个性化）
```

**优势：**
- 不需要为81型分别写完整独立文案（工作量大且难以维护）
- 12个维度模块（每个维度3个取值）+ 81个组合效应描述
- 总共需要：4×3=12个维度模块 + 81个组合简述 = 93个内容单元
- 远少于81篇完整描述的工作量

### 4.2 内容优先级

**需要优先完成的独立完整描述（核心16型）：**
传统16型 → 已在社区有成熟文案，翻译+本优化即可

**需要重点编写的组合描述（含1个中间值，共32型）：**
如 EABJ、INCD 等 → 这些是核心差异化内容

**可简化描述的类型（含2+个中间值，共33型）：**
如 ABCD、ABFJ 等 → 这类较少见，可用模块化拼接

### 4.3 描述文案质量标准

- [ ] 正面积极：强调优势，委婉表述成长空间
- [ ] 避免标签化：使用"倾向/偏好"而非"就是/必然是"
- [ ] 包含可操作建议：不只描述"是什么"，还提供"可以怎么做"
- [ ] 语言通俗：面向大众读者，不用心理学术语
- [ ] 文化适配：使用中国本土案例和生活场景
